# Oppimispäiväkirja


## Lukijalle
Oppimispäiväkirja on alun perin ollut erillinen Google Docs -tiedosto, jonne olen kirjoittanut ajatuksiani ja muistiinpanoja projektia tehdessä. Tämä on siistitty versio edellä mainitusta tiedostosta. Lisäykset ovat eri ohjelmointi kertoja, joiden aikoja en pistänyt ylös. Huom. kaikilta ohjelmointi kerroilta en ole laittanut mitään ylös, joten "päiväkirja" saattaa olla hieman "hyppivä".


## Lisäys 1
- Projektin luonti
- Free Dictionary API:n tutkiskelua


## Lisäys 2
- Peli- ja Kotisivujen luonti
- API hakujen kanssa säätämistä
    - API antaa haetun sanan tiedot kahteen kertaan
        - Syyksi ilmeni oma tyhmyys, eli kutsun API:a pelisivulla kahdesti, joka sekoitti vihjeiden haun.
    - API:n tiedot joillekin sanoille on puutteelliset
